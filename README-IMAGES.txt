IMAGES FOLDER - REQUIRED FILES
===============================

This folder must contain the following files before deploying the website:

1. bds-logo.png
   -------------
   - Company logo for Bad Debt Solutions
   - Recommended size: 200x200px or larger
   - Format: PNG (transparent background preferred)
   - Used in: Navigation bar and footer
   - Color: Should work well on both dark and light backgrounds

2. BDS_Client_Inquiry_Form.pdf
   ---------------------------
   - Client engagement form (PDF format)
   - This file is downloaded when users click "Engage Our Service"
   - Should contain form fields for client information
   - Professional formatting recommended

3. og-image.jpg
   ------------
   - Social media preview image
   - Recommended size: 1200x630px (Facebook/LinkedIn standard)
   - Format: JPG or PNG
   - Should represent Bad Debt Solutions professionally
   - This image appears when someone shares your website on:
     * Facebook
     * LinkedIn
     * WhatsApp
     * Twitter
   - Include company logo and tagline in the image

IMPORTANT NOTES:
----------------

- All file names are case-sensitive
- Use exact file names as listed above
- Maintain image quality (avoid over-compression)
- Test all files after uploading
- Ensure PDF is not password-protected
- Logo should be high resolution for retina displays

FILE PLACEMENT:
--------------

website-root/
  └── images/
      ├── bds-logo.png
      ├── BDS_Client_Inquiry_Form.pdf
      └── og-image.jpg

TESTING:
--------

After adding files, test:
✓ Logo appears in navigation (top left)
✓ Logo appears in footer
✓ PDF downloads when clicking "Engage Our Service"
✓ Social media preview shows correct image (use Facebook Debugger)
